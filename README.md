# Audio-to-Text-Conversion
The following methodology employed to solve the audio to text conversion.

1. ASR supported models are adopted.
2. Install all the dependency modules.
3. Whisper AI open source model is adopted.
4. Speaker diarization is applied to classify the speaker audio with time lemnth and text. 
5. Hugging face model is used.
6. The sorce code is a readily work for colab and the input files can be add from local drive data.
